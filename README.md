# asGiant-admin

- 基于egg架构的自动生成接口的框架

### 使用说明

> 你可以这么理解这个框架，就是在你配置完数据后，就生成了增删改查接口，也同时生成了相关的栏目。
> 这样即使你是一个不懂后端的同学，你也可以直接调用接口给前端使用

### 功能

- 增删改查数据库
- 自由添加字段
- 登录
- 权限
- 数据采集

### 技术栈

- egg
- mongdb
- avuex
